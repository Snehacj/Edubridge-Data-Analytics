* Name                
* Platform            
* Year_of_Release

* EU_Sales           
* JP_Sales           
* Other_Sales        
* Global_Sales       
* Critic_Score       
* Critic_Count       
* User_Score         
* User_Count
* Developer           
* Rating             
  
#### Conclusion :-
* Global sales of video game is maximum in Wii platform
